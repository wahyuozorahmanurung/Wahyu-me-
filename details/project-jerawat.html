Situation (Latar Belakang & Konteks Proyek)

Masalah kesehatan kulit wajah, khususnya jerawat, sering kali memicu penurunan rasa percaya diri bagi banyak orang. Sayangnya, keterbatasan akses, biaya konsultasi klinis, serta minimnya pengetahuan dasar membuat masyarakat kerap salah dalam mengidentifikasi jenis jerawat dan memilih penanganan yang tepat. Proyek ACNEVERSE ini diinisiasi sebagai proyek capstone dalam program mentereng DBS Foundation Coding Camp powered by Dicoding untuk menghadirkan solusi deteksi dini kondisi kulit yang mudah diakses, cepat, serta bernilai edukatif.

Task (Tantangan & Tanggung Jawab Tim)

Sebagai tim pengembang, kami bertanggung jawab penuh untuk merancang, melatih, dan mengimplementasikan model kecerdasan buatan yang mampu mengklasifikasikan multi-kelas jerawat secara presisi. Tantangan utamanya adalah melatih model Computer Vision agar sensitif mendeteksi berbagai variasi ukuran dan tipe jerawat langsung dari kamera pengguna. Selama proses pengerjaan, tim kami mendapatkan pendampingan intensif dari mentor profesional di bidang Artificial Intelligence, Machine Learning, Computer Vision, dan Software Engineering untuk memastikan standardisasi sistem memenuhi ekspektasi industri.

Action (Implementasi Teknis & Arsitektur Sistem)

Guna mewujudkan platform yang tangguh, responsif, dan fleksibel, kami membangun sistem ini dengan memisahkan fungsi antarmuka dan komputasi cerdas (Decoupled Architecture) lewat langkah strategis berikut:

1. Pengembangan Model Computer Vision (Deep Learning)

Kami melatih model YOLOv8 menggunakan dataset citra wajah yang dikurasi dan dianotasi secara manual. Model ini dioptimalkan khusus agar mampu mengenali serta memisahkan 7 kategori kondisi kulit secara akurat melalui penempatan bounding box dan nilai tingkat kepercayaan (confidence score):

  • Blackhead (Komedo Hitam) dan Whitehead (Komedo Putih)
  • Papules dan Pustules
  • Nodule
  • Bekas Jerawat dan Pori-Pori
2. Integrasi Full-Stack & Pembuatan API Service
  • FastAPI Backend Service: Berperan sebagai core mesin AI berbasis Python dan OpenCV yang bertugas menerima muatan gambar, melakukan proses inferensi model YOLOv8 secara real-time, lalu mengembalikan data koordinat serta label prediksi.
  • Laravel Frontend Web: Digunakan untuk merancang antarmuka pengguna, sistem autentikasi, serta logika navigasi program menggunakan bahasa PHP dan basis data MySQL.
  • Konektivitas REST API: Menjembatani komunikasi asinkronus antara gerbang Laravel dan klaster FastAPI agar proses kalkulasi gambar berjalan tanpa hambatan (seamless).
3. Rekayasa Fitur Berorientasi Pengguna

Kami menyusun alur penggunaan aplikasi yang sederhana dan ramah pengguna (Cross-Platform Mobile & Desktop Browser):

  • Pengguna membuka aplikasi ACNEVERSE via browser, lalu memilih opsi input via unggah file galeri atau tangkapan kamera real-time.
  • Gambar dikirim menuju server via REST API untuk dieksekusi oleh YOLOv8.
  • Sistem mengembalikan visualisasi posisi jerawat lengkap dengan detail karakteristik pemicu, edukasi penyebab, serta rekomendasi solusi perawatan kulit yang relevan.

Result (Hasil Akhir & Dampak)

Melalui kolaborasi tim yang solid dan bimbingan berkala, proyek ACNEVERSE sukses dirampungkan dan dipresentasikan dengan mulus di hadapan para mentor serta dewan evaluator program DBS Foundation Coding Camp. Sistem ini terbukti mampu mendeteksi letak dan mengklasifikasikan jenis jerawat secara instan dengan tingkat akurasi yang memuaskan. Keberhasilan integrasi teknologi YOLOv8, FastAPI, dan Laravel ini melahirkan produk teknologi kesehatan disruptif yang tidak hanya fungsional sebagai alat pemindaian wajah, tetapi juga sukses menjadi media edukasi interaktif guna meningkatkan kesadaran perawatan kulit masyarakat secara luas.
