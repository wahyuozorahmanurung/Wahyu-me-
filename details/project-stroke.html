Situation (Background)

Early identification of stroke is a crucial factor in the medical world to minimize the risk of patient mortality and permanent disability. However, the process of reading head CT Scan images manually requires time, high precision, and is heavily dependent on the availability of radiologists. In response to these challenges, this research project was initiated to build an artificial intelligence system based on Medical Image Analysis that can automatically, quickly, and objectively detect and classify Ischemic Stroke, Hemorrhagic Stroke, and Normal conditions.

Task (Research Challenges & System Target)

The primary challenge in this project is overcoming the visual complexity of medical imagery, where stroke lesions often have poorly defined boundaries and varying sizes. My main target is to design an integrated classification system architecture that is not only highly accurate but also interpretable (its decision-making process can be understood from a medical standpoint). Additionally, the system must accommodate various medical input data formats ranging from raw clinical data (DICOM) to standard image formats (PNG/JPG) to be processed into classification predictions ready to support clinical decisions (Decision Support System).

Action (Methodology & Technical Implementation)

To address these challenges, I designed and implemented a Two-Stage Classification System approach using a combination of deep learning technologies and advanced feature computing. The strategic steps taken include:

  • Stage 1 - Deep Learning-Based Localization (YOLOv11): Applying the cutting-edge object detection model YOLOv11 to intelligently detect, isolate, and extract relevant areas or Regions of Interest (ROI) indicated to have stroke lesions on CT Scan images.
  • Stage 2 - Texture Feature Extraction (Radiomics): The successfully isolated ROI areas are then processed using the PyRadiomics library. This stage manually extracts hundreds of high-level, non-visible texture features, such as intensity characteristics, shape, and texture matrices (GLCM, GLSZM).
  • Classification (Machine Learning): These quantitative features are then fed into a robust lineup of Machine Learning algorithms, such as SVM, Naive Bayes, and Random Forest using Scikit-Learn, which are trained and comparatively analyzed to find the best algorithm for determining the final 3-class classification outcome.
  • System Integration & API Development: Aligning the entire artificial intelligence pipeline into a production environment using Python and FastAPI, which is then integrated with a responsive web interface based on Laravel so that visual analysis results can be intuitively accessed by medical professionals.

Result (Final Outcome & Impact)

The implementation of this Two-Stage architecture successfully optimized system performance significantly compared to single-stage classification models. The combination of YOLOv11 and Radiomics analysis is capable of minimizing false-positives and providing a transparent quantitative foundation for physicians in analyzing stroke types. This system successfully represents a breakthrough in health technology that is capable of cutting diagnosis waiting times, providing precise visualization of lesion areas, and holds great potential to become a reliable digital assistant for radiologists in real clinical workflows.
